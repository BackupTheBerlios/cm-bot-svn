DT_cmd RMT_getCommand (  ) 

Liest empfangene Befehle vom Remote-Controller aus.

Instruction aus dem High- und Low-Teil zusammensetzen Bsp: Paket für B_2: 0 1 2 3 4 5 ... FF;55;20;DF;00;FF;FF;55;00;FF;00;FF; LL HH Eigentliche Information: 0x0020 d.h. H = Paket[4] L = Paket[2]
